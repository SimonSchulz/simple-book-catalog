# Book Catalog

## Task
Simple book catalog application with search, favorites, author filter, live search, and theme switcher using Open Library API.
[Task description](https://drive.google.com/file/d/1RBRcuH-_oAvtjem5Xs0c4NXZ8I38aYyH/view)

## How to run the app
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build`
4. Open `dist/index.html` in browser

## Folder Structure
- `src/` - JS source files and CSS
- `assets/` - images 
- `index.html` - main HTML file
- `dist/` - production build (HTML, JS, assets)
